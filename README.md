🐾 PetLove — Платформа для пошуку улюбленців 🐶🐱
🚀

PetLove — це сучасна та швидка платформа для тих, хто шукає вірного друга або хоче допомогти тваринам знайти новий дім. Проєкт створено з любов'ю до деталей та використанням передових технологій. ❤️

✨ Унікальні фічі проєкту

🌓 Theme Switcher (Cat & Dog Mode): - 🐕 Dog Mode — енергійна світла тема.

🐈 Cat Mode — затишна темна тема.

🌍 Multi-language (UA/EN): Повна підтримка двох мов для зручності кожного користувача. 🇺🇦 en

🔍 Smart Search: Пошук за ключовим словом, статтю, типом та локацією.⚡

🧪 E2E Testing: Весь критичний функціонал пошуку та фільтрації протестовано за допомогою Cypress. ✅

👤 User Profile: Керування власними оголошеннями, редагування профілю та список "Обраного". 🦴

🛠 Стек технологій
Framework: React 19 ⚛️

Build Tool: Vite ⚡

State Management: Redux Toolkit 🏪 (RTK Query для API 🌐)

Language: TypeScript 📘

Styling: CSS Modules 🎨

Validation: React Hook Form + Yup ✅

Testing: Cypress 🧪

Internationalization: i18next 🗺

🏗 Архітектура (FSD)
Проєкт побудований за методологією Feature-Sliced Design. Це робить код масштабованим та легким у підтримці. 🧩

app/ ⚙️ — ініціалізація, провайдери та глобальні стилі.

pages/ 📄 — компоненти сторінок (Home, News, Notices, Friends, Profile).

widgets/ 🧱 — складні блоки (Header, Filters, UserCard).

features/ 🛠 — бізнес-логіка (Auth, ThemeSwitcher, LanguageSwitcher).

entities/ 🐕 — сутності (Notice, Pet, News, Friend).

shared/ 🛠 — UI-kit (Button, Input, Loader), хуки та API-клієнт.

🚀 Як запустити локально
Клонуйте репозиторій: 📥

git clone https://github.com/valentina2209/petlove

Встановіть залежності: 📦

npm install

Запустітm проєкт: 🚀

npm run dev

Запустіть тести: 🧪

npm run cypress:open

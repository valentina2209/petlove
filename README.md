# 🐾 PetLove — Платформа для пошуку улюбленців 🐶🐱

🚀 **PetLove** — сучасна платформа для тих, хто шукає вірного друга або хоче допомогти тваринам знайти новий дім.
Проєкт створено з любов’ю до деталей та використанням сучасних технологій. ❤️

---

## ✨ Основні можливості

### 🌓 Theme Switcher — Cat & Dog Mode

Перемикач тем для комфортного користування:

- 🐕 **Dog Mode** — енергійна світла тема
- 🐈 **Cat Mode** — затишна темна тема

---

### 🌍 Multi-language (UA / EN)

Повна підтримка двох мов для зручності користувачів.

UA Українська
EN English

---

### 🔍 Smart Search

Зручний пошук тварин за різними параметрами:

- ключове слово
- стать
- тип тварини
- локація

---

### 👤 User Profile

Користувач може:

- керувати власними оголошеннями
- редагувати профіль
- додавати тварин до **Favorites**

---

### 🧪 E2E Testing

Критичний функціонал перевірений за допомогою **Cypress**:

- пошук
- логінізаці
- реєстрація
- взаємодія з формами

# 🏗 Архітектура проєкту

Проєкт побудований за методологією **Feature-Sliced Design (FSD)**.
Це дозволяє зробити код масштабованим та зручним для підтримки.

```
src/

app/        ⚙️ Ініціалізація, провайдери, глобальні стилі
pages/      📄 Сторінки (Home, News, Notices, Friends, Profile)
widgets/    🧱 Складні UI блоки (Header, Filters, UserCard)
features/   🛠 Бізнес-логіка (Auth, ThemeSwitcher, LanguageSwitcher)
entities/   🐾 Сутності (Notice, Pet, News, Friend)
shared/     🧰 UI-kit, хуки, утиліти та API-клієнт
```

# 🚀 Запуск проєкту локально

### 1️⃣ Клонувати репозиторій

```
git clone https://github.com/valentina2209/petlove
```

### 2️⃣ Перейти в папку проєкту

```
cd petlove
```

### 3️⃣ Встановити залежності

```
npm install
```

### 4️⃣ Запустити проєкт

```
npm run dev
```
